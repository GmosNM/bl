* Stack operations for call instruction  

  pc   program counter (pointer to current instruction)
  CFS  current frame stack (used later for rollback of the stack)

  call fn (1, 2, 3) 4

    | stack op | data         | instr |
    |----------+--------------+-------|
    | PUSH     | 3            | ?     |
    | PUSH     | 2            | ?     |
    | PUSH     | 1            | ?     |
    | PUSH CFS | pc, prev CFS | call  |
    | ...      | -            | -     |
    | POP CFS  | -            | ret   |
    | POP      | -            | ret   |
    | POP      | -            | ret   |
    | POP      | -            | ret   |
    | PUSH     | 4            | ret   |
